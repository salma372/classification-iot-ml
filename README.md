# Classification de flux réseau IoT par Machine Learning

## Contexte
Projet personnel appliqué au dataset CIC-IoT2023. L'objectif est de traiter des données réelles de cybersécurité

# Objectif 
-Nettoyer et préparer le dataset
-Entrainer un modèle RandomForest
-EAvaluer les performances du modèle
-Sauvegarder le modèle afin de pouvoir le réutiliser 

>Il s'agit d'une première version du pipeline. Des améliorations comme l’ingénierie de features, le tuning des hyperparamètres et l’ajout de visualisations des performances pourront être ajoutées

## Donnée
-Dataset CIC-IoT2023
